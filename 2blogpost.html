The Innovation of HTML5

HTML5 is a new web developer 14 years after HTML4 has been released. Some key changes that W3C focuses are this new version of HTML is to make the website runs smoother and more consistent. HTML5 aims to make the whole system to be more simplified with some new attributes. Before having HTML5, web pages need HTTP cookies to send the user’s data. However, in HTML5, they eliminate the need of HTTP cookies.

In HTML5, W3C improves error handling and the function of replacing the script with more markups. The new attributes they added in HTML5 include the function to allow video control of play, pause, and volume options without additional plugins. Some other new things for web users include more options for displaying fonts, colors, and effects. Also, the browser can display 3D graphics just by using the computer’s original graphics processor.

In conclusion, the new HTML5 lets more dynamic content to deliver faster through the webpage. On the other hand, HTML5 also has some new features for the web developers. For example, maximizing the support of mobile services. Nowadays, the uses of mobile devices have become much more common. Therefore, HTML5 now provides support to mobile users. HTML5 also gives the developers more control of the language of Flash and JS-based hacks.
